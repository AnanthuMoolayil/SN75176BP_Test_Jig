# SN75176BP_Test_Jig
Test Jig to verify standard operating capability of SN75176BP ICs using Arduino UNO.

---

## Features:
1. Capable of testing 10 ICs at once.
2. LED ON condition indicates IC working is OK.

---

## Major Components
1. Arduino UNO
2. Shift registers 74HC595
3. Multiplexer board CD74HC4067<br>
*Note: BOM provided.*
